# 祖名闪电说 UI Demo

这是设计交接用的可点击原型，不包含真实录音、阿里云调用、SQLite 或跨应用写入。

## 可演示内容

- 首页与设置页切换。
- 历史记录复制、删除、撤销、更多菜单和详情抽屉。
- 模拟播放录音和重新转写。
- 阿里云连接与麦克风测试反馈。
- 口语顺滑、备用热键和自动写入模式切换。
- 胶囊默认隐藏；在浏览器中按右 Alt 后出现“直接说”。
- 再次按右 Alt 模拟“识别中 → 已写入 → 自动消失”，按 Esc 模拟取消。

## 本地运行

```powershell
cd prototype
npm install
npm run dev
```

## 生产边界

此原型只能作为 UI 和交互依据。最终 Windows 应用必须使用 .NET 10 WPF，并按 `docs/DEVELOPMENT_HANDOFF.md` 实现录音、阿里云识别和分层自动写入。

网页原型无法真正悬浮到其他 Windows 应用上方。生产版本必须把胶囊实现为独立 Topmost、NoActivate、无任务栏图标的悬浮窗口；主窗口最小化或收进系统托盘时仍可使用。

生产版本中，右 Alt 在任何页面都能开始听写。若没有输入框，识别结果正常保存到应用历史，胶囊显示“已保存”；这不属于写入失败。
